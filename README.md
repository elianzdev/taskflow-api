# TaskFlow

Aplicación Full Stack para gestión de tareas desarrollada con React, Spring Boot y PostgreSQL.

---

## Tecnologías utilizadas

### Frontend
- React
- React Router
- Axios
- Bootstrap
- Vite

### Backend
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Maven

### Base de datos
- PostgreSQL

---

## Features

- Registro de usuarios
- Login con JWT
- Protección de rutas
- CRUD completo de tareas
- API REST segura
- Arquitectura en capas
- Validaciones backend
- Manejo global de errores

## Instalación

### Clonar repositorio

```bash
git clone TU_REPO
```

---

## Backend

Entrar al backend:

```bash
cd backend
```

Ejecutar:

```bash
.\mvnw.cmd spring-boot:run
```

Servidor:

```bash
http://localhost:8080
```

---

## Frontend

Entrar al frontend:

```bash
cd frontend
```

Instalar dependencias:

```bash
pnpm install
```

Ejecutar:

```bash
pnpm dev
```

Servidor:

```bash
http://localhost:5173
```
